# Style Guide: WordPress Translator App (Wapuulingo)

## 1. Visual Identity
The visual language is **bold, gamified, and high-contrast**, designed to make translation tasks feel like a rewarding game.

### Color Palette
| Element | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary Navy** | `#003c56` | Main backgrounds, headers, and nav bars. |
| **Mascot Yellow** | `#fecb00` | Primary CTAs, mascot body, and highlights. |
| **Success Green** | `#4ab866` | Completion states, Submit buttons, and progress. |
| **Neutral White** | `#ffffff` | Content cards, input fields, and body text on dark. |
| **Status Grey** | `#e0e0e0` | Inactive states or background for text fields. |

---

## 2. Typography
* **Font:** Modern Sans-Serif (e.g., *Inter*, *SF Pro*, or *Roboto*).
* **Headlines:** Bold or Extra Bold. Used for titles and big numbers.
* **Body:** Regular or Medium. Used for labels and instructions.
* **Stats:** Large, prominent font sizes for "Translated" and "Approved" counts to drive user motivation.

---

## 3. UI Components
### Cards & Containers
* **Radius:** 12px to 16px (Rounded).
* **Style:** Flat design. Avoid heavy drop shadows; use color contrast to define depth.
* **Layout:** Vertical stacking with generous padding (approx. 20px).

### Buttons & Inputs
* **Primary Button:** Full width, high-contrast background (Yellow or Green).
* **Input Fields:** Subtle grey background with a colored border (Green) when active.
* **Progress Bar:** Thin, horizontal bar at the top of the screen to indicate session completion.

---

## 4. Illustration Style (Wapuu)
* **Character:** The official "Wapuu" mascot.
* **Line Work:** Thick, consistent black outlines (3pt - 5pt).
* **Coloring:** Flat colors; minimal, hard-edged shading.
* **Expressions:** Always positive, enthusiastic, or focused.
* **Placement:** * **Hero:** Large center-piece on the home screen.
    * **Feedback:** Small "peeking" versions for success messages or string headers.

---

## 5. Voice & Tone
* **Supportive:** "Great job! More approvals are on the way."
* **Direct:** "Translate to: Portuguese (Brazil)."
* **Gamified:** Use terminology like "XP," "Points Awarded," and "Approved."
* **Concise:** Keep string labels short and actionable.
